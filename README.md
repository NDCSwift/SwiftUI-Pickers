# 🎛️ SwiftUI Pickers

A SwiftUI demo covering `DatePicker`, `ColorPicker`, and `Picker` (option/segmented) — showing how to bind each picker type to state and customize their appearance.

---

## 🤔 What this is

This project demonstrates the three main picker controls in SwiftUI: `DatePicker` for selecting dates and times, `ColorPicker` for choosing colors, and `Picker` for selecting from a list of options with wheel, segmented, and menu styles. Each example is wired to a `@State` variable and rendered live.

## ✅ Why you'd use it

- **DatePicker** — date/time selection with `displayedComponents` control
- **ColorPicker** — native color wheel bound to a `Color` state variable
- **Picker styles** — `.segmented`, `.wheel`, and `.menu` display styles compared side by side
- **@State binding** — all pickers demonstrate proper two-way binding patterns

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUI-Pickers.git
cd SwiftUI-Pickers
```

### 2. Open in Xcode
- Double-click `SwiftUI Pickers.xcodeproj`

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team**

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier (e.g., `com.yourname.SwiftUIPickers`).

---

## 🛠️ Notes

- If you see a code signing error, check that Team and Bundle ID are set.

## 📦 Requirements

- iOS 16+
- Xcode 15+
- Swift 5.9+
